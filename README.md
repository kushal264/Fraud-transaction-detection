# Fraud-transaction-detection
Machine Learning based fraud detection system with a Streamlit web interface for real-time transaction prediction.


This project predicts fraudulent financial transactions using Machine Learning.

## Tech Stack
- Python
- Scikit-learn
- Streamlit
- Pandas

## Features
- Predicts whether a transaction is fraud or legitimate
- Interactive UI using Streamlit
- Trained ML model pipeline

## Run Locally

pip install -r requirements.txt
streamlit run app.py
